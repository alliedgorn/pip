## What Happened

This was the biggest QA session in Den history. 28 code reviews in one sitting — more than some Beasts ship in a week. The work spanned DM overlays, Done column lazy loading, Prowl features (datetime, Telegram notifications, advance reminders), Withings OAuth integration, Forge body composition, API help endpoint, and multiple CSS fixes.

## Key Findings

1. **Timezone bug (T#475)** — Found that Prowl due dates stored in local time were compared against UTC, making notifications fire 7 hours late. Root cause analysis with actual datetime comparison proved the issue.

2. **Content-length header bug (T#476)** — Hono's c.body() with Buffer sends content-length: 0. Combined with Cloudflare's immutable caching, this served empty JS bundles to Gorn.

3. **CF cache blocking everything** — Multiple frontend features shipped but invisible to Gorn due to stale Cloudflare cache. Diagnosed by curling the actual bundle from the server.
