## What I Learned

1. **Always check composite keys before crying duplicate** — name alone isn't enough, check all uniqueness dimensions
2. **useEffect with deps fires on mount** — when reviewing React code, always ask "what happens on initial mount?" not just "what happens when this dep changes"
3. **Review speed vs depth tradeoff** — 14 reviews is a lot. The T#443 miss happened because I reviewed the logic but didn't simulate the mount lifecycle mentally
