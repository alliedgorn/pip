## Lessons Learned

1. **Check DB-level correlation, not just code paths.** BUG-3 was only findable by actually querying both tables and checking if the promised correlation column existed. Code review alone wouldn't catch it — the code was correct, the schema was incomplete.
2. **Alert threshold testing requires volume.** Single-event tests don't exercise the threshold logic. Had to fire 5 rapid failures from the same IP to trigger `alert_triggered`. Good pattern for future security QA.
