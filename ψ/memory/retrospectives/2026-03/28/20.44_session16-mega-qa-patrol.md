## What I Learned

1. **Live QA during active dev is chaos but effective** — found bugs faster when testing alongside the build cycle rather than after
2. **Screenshots from Gorn are gold** — the workout trends bug and delete-without-confirm were both caught from Gorn's screenshots. Always check uploaded images.
3. **Data anomalies aren't always code bugs** — Lateral Raises 20kg→56kg jump was likely import data, not a code issue. Don't file everything as a bug.
